🎤 Assistente de Voz com IA (Python)

Este projeto implementa um assistente de voz inteligente utilizando tecnologias modernas de Inteligência Artificial.

🚀 Funcionalidades
🎤 Captura de áudio pelo microfone
📝 Conversão de fala para texto (Speech-to-Text)
🤖 Processamento com IA (ChatGPT)
🔊 Conversão de texto para fala (Text-to-Speech)
🧠 Tecnologias utilizadas
Python
OpenAI Whisper
OpenAI ChatGPT API
gTTS (Google Text-to-Speech)
Google Colab
🔄 Fluxo da aplicação
Voz → Texto → IA → Resposta → Voz
▶️ Como executar
Abra o projeto no Google Colab
Insira sua chave da OpenAI:
os.environ['OPENAI_API_KEY'] = 'SUA_API_KEY'
Execute todas as células
Permita o uso do microfone
Fale algo e aguarde a resposta em áudio
💡 Possíveis melhorias
Interface gráfica com Streamlit
Suporte a múltiplos idiomas
Respostas mais rápidas com modelos otimizados
Integração com APIs externas
